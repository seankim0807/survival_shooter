A simple top-down survival shooter made in Godot 4, where all game logic is written in one script file for learning and experimentation.

Survive as long as possible while enemies chase you down. Shoot them to gain score and try to beat your high score.

✨ Features

Player movement (top-down)

Shooting system

Enemy spawning & chasing AI

Collision detection

Score tracking

Game over & restart

Entire game logic in a single script file
